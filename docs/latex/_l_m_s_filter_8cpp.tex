Este archivo contiene la implementación de la clase \doxylink{class_l_m_s_filter}{LMSFilter}, que actúa como wrapper C++ para las funciones de filtrado LMS (Least Mean Squares) adaptativo de CMSIS-\/\+DSP. La implementación se enfoca en\+:
\begin{DoxyItemize}
\item Adaptación automática de coeficientes para minimizar el error cuadrático medio
\item Gestión eficiente de memoria para estados y coeficientes adaptativos
\item Interfaz simplificada para procesamiento adaptativo de bioseñales
\item Optimización para procesadores ARM Cortex-\/M con SIMD
\end{DoxyItemize}

El algoritmo LMS implementado realiza\+:
\begin{DoxyEnumerate}
\item Filtrado FIR con coeficientes actuales\+: y\mbox{[}n\mbox{]} = Σ(w\mbox{[}k\mbox{]} \texorpdfstring{$\ast$}{*} x\mbox{[}n-\/k\mbox{]})
\item Cálculo del error\+: e\mbox{[}n\mbox{]} = d\mbox{[}n\mbox{]} -\/ y\mbox{[}n\mbox{]}
\item Actualización de coeficientes\+: w\mbox{[}k\mbox{]} = w\mbox{[}k\mbox{]} + μ \texorpdfstring{$\ast$}{*} e\mbox{[}n\mbox{]} \texorpdfstring{$\ast$}{*} x\mbox{[}n-\/k\mbox{]}
\end{DoxyEnumerate}
