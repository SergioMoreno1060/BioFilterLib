Este archivo contiene la implementación de un wrapper C++ para los filtros IIR de CMSIS-\/\+DSP, diseñado para el procesamiento eficiente de bioseñales en Arduino Due. La clase \doxylink{class_i_i_r_filter}{IIRFilter} facilita el uso de filtros IIR (Infinite Impulse Response) manteniendo las optimizaciones de hardware de CMSIS-\/\+DSP.
