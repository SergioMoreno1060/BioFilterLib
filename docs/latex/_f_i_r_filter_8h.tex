Este archivo contiene la implementación de un wrapper C++ para los filtros FIR de CMSIS-\/\+DSP, diseñado específicamente para el procesamiento de bioseñales en Arduino Due. La clase \doxylink{class_f_i_r_filter}{FIRFilter} facilita el uso de filtros FIR manteniendo las optimizaciones de hardware que ofrece CMSIS-\/\+DSP para procesadores ARM Cortex-\/M.

\begin{DoxyNote}{Note}
Requiere CMSIS-\/\+DSP y está optimizado para Arduino Due (ARM Cortex-\/\+M3) 
\end{DoxyNote}
