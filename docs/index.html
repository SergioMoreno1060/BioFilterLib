BioFilterLib

Librería en C/C++ para filtros digitales (FIR, IIR, y más)

Instalación Ejemplo de Uso Estructura GitHub

Instalación

Para usar la librería en Arduino IDE:

  1. Descarga este repositorio como .zip.
  2. En Arduino IDE, ve a Programa → Incluir librería → Añadir biblioteca .ZIP.
  3. Selecciona el archivo descargado.

Ejemplo de Uso

Un ejemplo básico de uso con un filtro FIR:

#include <BioFilterLib.h>

FIRFilter myFilter;

void setup() {
  Serial.begin(9600);
  float coeffs[3] = {0.25, 0.5, 0.25};
  myFilter.init(coeffs, 3);
}

void loop() {
  float input = analogRead(A0);
  float output = myFilter.process(input);
  Serial.println(output);
  delay(10);
}

También puedes ver el ejemplo completo en: /examples/FIR_Example/FIR_Example.ino.
