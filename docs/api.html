FIRFilter src/filters/FIRFilter.h

Wrapper C++ para arm_fir_instance_f32 (CMSIS-DSP). Filtro de Respuesta al Impulso Finita con fase lineal garantizada. Procesa float32_t.

FIRFilter(float32_t* coeffs, uint16_t numTaps, uint16_t blockSize)
CONSTRUCTOR
PARÁMETROTIPODESCRIPCIÓN
coeffsfloat32_t*Array de coeficientes. Debe mantenerse válido mientras exista el objeto.
numTapsuint16_tNúmero de coeficientes (= orden + 1). Rango recomendado: 11–101.
blockSizeuint16_t1 para tiempo real. Potencias de 2 (32, 64…) para batch. Estado interno: numTaps + blockSize - 1 floats.
float32_t processSample(float32_t input)
MÉTODO

Aplica el filtro FIR a una sola muestra. Mantiene el estado interno automáticamente. Ideal para el loop().

Entrada:Muestra a filtrar (normalmente en [-1.0, 1.0])
Retorna:Muestra filtrada float32_t
void processBuffer(float32_t* input, float32_t* output, uint32_t length)
MÉTODO

Procesa un array completo. Mayor throughput por instrucciones SIMD del Cortex-M3. Los arrays no deben solaparse.

PARAMDESCRIPCIÓN
inputArray de entrada (mínimo length elementos)
outputArray de salida (mínimo length elementos, no solapar con input)
lengthNúmero de muestras a procesar
IIRFilter src/filters/IIRFilter.h

Wrapper C++ para arm_biquad_casd_df1_inst_f32 (CMSIS-DSP). Filtro IIR en cascada de secciones biquad (Direct Form 1). Alta selectividad con bajo costo computacional.

IIRFilter(float32_t* coeffs, uint8_t numStages, uint16_t blockSize)
CONSTRUCTOR
PARÁMETROTIPODESCRIPCIÓN
coeffsfloat32_t*Array de numStages × 5 coeficientes en orden {b0, b1, b2, a1, a2}. a1 y a2 con signo invertido respecto a scipy/MATLAB.
numStagesuint8_tNúmero de secciones biquad. Un filtro de orden $2M$ necesita $M$ etapas. Memoria: 4 × numStages floats.
blockSizeuint16_tUsar 1 para tiempo real.
float32_t processSample(float32_t input)
MÉTODO

Filtra una muestra individual. Retarda de grupo mínimo. Idéntica firma que FIRFilter::processSample().

void processBuffer(float32_t* input, float32_t* output, uint32_t length)
MÉTODO

Procesa un array de muestras. Los arrays no deben solaparse.

⚠ Signo de coeficientes a: CMSIS-DSP espera $a_1$ y $a_2$ con signo negado respecto a la ecuación estándar y respecto a scipy.signal. Multiplicar por -1 al exportar desde Python.
LMSFilter src/filters/LMSFilter.h

Wrapper C++ para arm_lms_norm_instance_f32 (CMSIS-DSP). Implementa el algoritmo LMS Normalizado (NLMS). Los coeficientes se actualizan automáticamente durante el procesamiento.

LMSFilter(float32_t* coeffs, uint16_t numTaps, float32_t mu, uint16_t blockSize)
CONSTRUCTOR
PARÁMETROTIPODESCRIPCIÓN
coeffsfloat32_t*Coeficientes iniciales. Inicializar a 0.0f. Serán modificados durante la adaptación.
numTapsuint16_tNúmero de coeficientes. Rango típico: 16–128. Memoria de estado: numTaps floats.
mufloat32_tPaso de adaptación. ECG: 0.01–0.05 · EMG: 0.001–0.01 · EEG: 0.0001–0.001. Valores grandes = convergencia rápida pero inestable.
blockSizeuint16_tUsar 1 para tiempo real.
void processSample(float32_t input, float32_t reference, float32_t* output, float32_t* error)
MÉTODO
PARÁMETRODESCRIPCIÓN
inputSeñal primaria contaminada (ECG + ruido)
referenceSeñal de referencia del ruido (ej: seno de 60 Hz, canal EOG)
outputPuntero → señal limpia resultante
errorPuntero → error de adaptación (reference − output). Decrece al converger.
void processBuffer(float32_t* input, float32_t* reference, float32_t* output, float32_t* error, uint32_t length)
MÉTODO

Procesa un bloque completo. Los coeficientes se actualizan continuamente dentro del bloque. Todos los arrays deben tener al menos length elementos.

float32_t getMu() const
void setMu(float32_t newMu)
void resetCoefficients(const float32_t* newCoeffs = nullptr)
MÉTODOS

getMu() — retorna el valor actual de μ.

setMu(newMu) — ajusta μ en tiempo real sin reinicializar el filtro. Útil para esquemas de μ variable.

resetCoefficients(newCoeffs) — reinicia los pesos. Si newCoeffs = nullptr, los inicializa a cero. Usar si el filtro diverge o al cambiar de señal.

WaveletFilter src/filters/WaveletFilter.h

Banco de 4 filtros FIR que implementa la Transformada Wavelet Discreta con wavelet Daubechies-4 (8 coeficientes por filtro). Coeficientes predefinidos — no requiere cálculo externo.

WaveletFilter(uint16_t blockSize)
CONSTRUCTOR

Inicializa los 4 filtros FIR internos (análisis LP, análisis HP, síntesis LP, síntesis HP) con los coeficientes DB-4. Usar blockSize=1 para tiempo real.

void processSample(float32_t input, float32_t* approxCoeff, float32_t* detailCoeff)
MÉTODO
PARÁMETRODESCRIPCIÓN
inputMuestra de entrada
approxCoeffPuntero → coeficiente de aproximación (baja frecuencia, ≤ Fs/4)
detailCoeffPuntero → coeficiente de detalle (alta frecuencia, Fs/4 – Fs/2)
void processBuffer(float32_t* input, float32_t* approxArray, float32_t* detailArray, uint32_t length)
MÉTODO

Versión por bloques. Mayor throughput. Los 3 arrays deben tener al menos length elementos y no solaparse.

float32_t reconstruct(float32_t approxCoeff, float32_t detailCoeff)
MÉTODO

Síntesis wavelet. Combina aproximación y detalle para reconstruir la señal.

reconstruct(approx, detail) → señal original
reconstruct(approx, 0.0f) → señal suavizada (sin ruido HF)
reconstruct(0.0f, detail) → solo bordes y transitorios
void reset()
MÉTODO

Limpia el estado de los 4 filtros FIR internos. No modifica los coeficientes DB-4. Usar al cambiar de paciente o reiniciar sesión.

utils src/utils/utils.h

Funciones C para métricas de señal. Incluidas con extern "C" para compatibilidad C++.

FUNCIÓN FIRMA DESCRIPCIÓN
calculateSNR float(signal*, noise*, uint32_t) Relación señal-ruido en dB. $\text{SNR} = 10\log_{10}(P_s/P_n)$
calculateMSE float(original*, filtered*, uint32_t) Error cuadrático medio entre dos señales
calculateRMS float(signal*, uint32_t) Valor eficaz (Root Mean Square) de la señal
calculateStdDev float(signal*, uint32_t) Desviación estándar de la señal
calculateCorrelation float(a*, b*, uint32_t) Coeficiente de correlación de Pearson entre dos señales. Rango: [-1, 1]
